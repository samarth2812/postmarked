# Codex Rules

Before making changes:

Read:

- 00-Vision.md
- design-system.md

Never redesign the UI.

Treat exported Figma screens as the source of truth.

Implement one milestone at a time.

Do not continue past the requested milestone.

Keep components modular.

Keep files small.

Explain architectural decisions.

Never replace working code.

Optimize for readability.

Optimize for performance.

Prefer reusable systems.

Use React Three Fiber for 3D rendering.

Use Framer Motion for UI animation.

Use GSAP only when timeline control is needed.

Avoid unnecessary React renders.

Maintain 60 FPS wherever possible.
