# Project Vision

## Name

Postmarked

## Tagline

Collected over time.

## Goal

Build an emotional interactive archive of memories.

Users explore a map.

Each place contains memories.

Each memory is represented as a Polaroid.

The experience should feel:

- Calm
- Cinematic
- Interactive
- Premium
- Emotional

Never flashy.

Motion should communicate meaning.

Every interaction should feel intentional.

This is not a CRUD application.

This is an experience.
